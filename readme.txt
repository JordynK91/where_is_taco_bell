Welcome to Where is Taco Bell, America's favorite command line interface for finding open Taco Bells near you.


If you are not me and cannot access the .env file, please use the API key I have provided you.
If I have not given you an API key, get your own from https://fusion.yelp.com/

To use the program, please run the following commands:
`python3 -m venv venv`
`source venv/bin/activate`
`pip install -r requirements.txt`
`python main.py`
